※このアプリケーションはインターフェースがありません。ブラウザのコンソールを起動して操作してください。

現在、コントラクトの動作は完璧になっています。コントラクトは、jsのwindow.instance変数に格納されています。
Ethereumネットワークに接続するために、拡張機能であるMetaMaskをブラウザにインストールしてください。

起動手順
1. このリポジトリをクローンしたディレクトリに移動。
2. npm install
3. npm -g install truffle
4. npm -g install ganache-cli
5. ganache-cli（ここでプロンプトは戻ってこないので新しくターミナルを起動してください。）
6. truffle migrate
7. このページのブラウザコンソールを開き、window.instance.メソッド名　を叩くことで、ブロックチェーンにデータのコミット・リードが行えます。
8. また、ブラウザをリロードすると、ブラウザコンソール内にアスキーアートで掲示板を表示します。
